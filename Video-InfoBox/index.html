With link

Without link

Available attributes:

  • height - specify height in px

Excluded attributes will default to the following:

  • height - 400px

Required Elements

The <video-inbox> element must contain the following child elements:

  • <h2> - This will be the heading for the info box.
  • <p> - This will be the paragraph of text that follows the heading.

Optional Elements

The <video-inbox> element can contain the following optional child elements:

  • <a> - The provided URL will be used as for a button following the paragraph of text in the info box.

HTML

<video-infobox src="https://www.w3schools.com/howto/rain.mp4" height="500px">
    <h2>Title</h2>
    <p>Description of video or content.</p>
    <a href="https://google.com"></a>
</video-infobox>
